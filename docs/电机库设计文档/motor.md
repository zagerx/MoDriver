# 电机状态

## 使能和失能
- 使能和失能分别对应PWM的开启和关闭，是开关量不能将其定义为状态

## 常见的状态
1. 故障态
     - 过流/欠压
     - 缺相/编码器故障
    
   允许根据故障类型设置更新不同的占空比

2. 运行态

   根据模式来使用不同的环路，比如轮廓路径模式，速度模式，回零模式。无论使用什么模式，该状态的边界为读取当前位置/速度/电流，调用不同的环路及策略，输出相应的站空比。
   
   - 进入runing状态，必须对用到的参数/算法进行初始化

3. 空闲态

   该状态下持续设置三相占空比为0.5

4. 校准态

   该状态下允许根据不同的策略来调整设置不同的占空比。
   - 电流偏置点测量
   - 编码器校准
   - 电机本体参数校准
